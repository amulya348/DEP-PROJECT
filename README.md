 📊 EduInsight – Student Performance Analysis System

📌 Overview

EduInsight is a simple and interactive web-based project designed to analyze student performance using multiple academic factors. It helps in evaluating students based on marks, attendance, LMS usage, and assignment completion.

The system calculates a performance score and classifies students into categories such as **High Performer**, **Average**, or **At Risk**.

 🎯 Objectives

* Analyze student performance using multiple parameters
* Identify patterns affecting academic success
* Provide instant performance evaluation
* Classify students into performance categories
* Create a simple and user-friendly interface

 🛠️ Technologies Used

* HTML
* CSS (optional for styling)
* JavaScript (for logic and calculations)
* VS Code (Development Environment)

 🚀 Features

* User-friendly input form
* Real-time performance calculation
* Instant result display
* Lightweight and fast (no backend required)
* Easy to run using Live Server

📊 How It Works

1. User enters:

   * Marks
   * Attendance
   * LMS Usage
   * Assignments
2. System calculates performance score using weighted formula:

   * Marks (40%)
   * Attendance (20%)
   * LMS Usage (20%)
   * Assignments (20%)
3. Based on score, student is classified into:

   * High Performer
   * Average
   * At Risk


 ▶️ How to Run the Project

 Option 1: Using VS Code (Recommended)

1. Open project folder in VS Code
2. Right-click on `index.html`
3. Click **"Open with Live Server"**

Option 2: Without Live Server

1. Open `index.html` directly in browser
2. Enter values and click **Calculate**
